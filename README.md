# Studyboard-Distributed-Uni-SWA201-1-Practice-3
Это распределённая версия учебной системы задач. Пользователь работает через отдельный клиентский интерфейс, задачи хранятся и изменяются через микросервис задач, события изменения статуса передаются сервису событий, который вызывает serverless-функцию, а часть изменений может передаваться другому узлу по P2P-каналу.
